# Вариант 2

**Лабораторная работа №10 — Веб-разработка: FastAPI (Python) vs Gin (Go)**

**Цель:** сравнить подходы к созданию веб-сервисов на Python и Go, реализовать взаимодействие между сервисами на разных языках.

**Теоретические положения:**

- Python: FastAPI, асинхронность, Pydantic
- Go: Gin/Echo фреймворки, middleware, контексты
- Взаимодействие: gRPC, REST, message brokers (RabbitMQ)

**Ссылки:**

- Gin: <https://gin-gonic.com/>
- gRPC: <https://grpc.io/>

# Задания

## Средней сложности

### Задание 2

Добавить middleware для логирования в Go.

### Задание 4

Создать FastAPI-сервис, который вызывает Go-сервис через HTTP.

### Задание 6

Сравнить скорость ответа FastAPI и Gin под нагрузкой (wrk/ab).
