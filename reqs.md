Правила сдачи лабораторных работ

Как сдавать:
Отправляете ссылку на Git-репозиторий. Вашу работу проверяет AI-агент-ревьюер, который автоматически:

1. Проверяет атомарные комиты и осмысленную историю комитов.
2. Проверяет наличие и содержание PROMPT_LOG.md.
3. Делает code review кода.
4. Выдаёт результат по каждому заданию + обратную связь.
5. Проверяет наличие юнит-тестов и их корректность.

Работа НЕ принимается к проверке, если:
— Код не в Git-репозитории или нет осмысленной истории коммитов
— Нет юнит-тестов или есть красные юнит-тесты
— Нет файла PROMPT_LOG.md
— Prompt log скопирован у другого студента — работа аннулируется целиком

Порядок сдачи:
Задания повышенной сложности выполняются только после выполнения всех заданий средней сложности.

Повторная отправка:
В этой девятой лабораторной повторная отправка разрешена — можно исправить замечания агента и отправить заново. Начиная со следующих лабораторных работ повторной отправки не будет — сдаёте один раз, управляйте временем.

Code review:
В девятой лабе агент укажет на критические проблемы в коде, но задание всё равно будет принято. Начиная со следующих лабораторных наличие критических проблем = задание не принято.

Правила:
— Можно использовать любой AI-инструмент (или несколько)
— Git обязателен с осмысленной историей комитов
— Один раз за семестр можно поменять вариант одной лабораторной работы
— Можно сдать работы заранее
— Для каждой лабораторной работы отдельный репозиторий

Требования к Prompt Log
Создайте файл PROMPT_LOG.md в корне репозитория. Документируйте каждое задание.
Шаблон:

# Prompt Log

## Задание М1: Менеджер задач

### Промпт 1

**Инструмент:** Claude Code
**Промпт:** "Создай модуль task_manager.py с классами Task и TaskManager..."
**Результат:** Получил базовую реализацию, но save_to_json не сериализовал даты.

### Промпт 2

**Промпт:** "Исправь сериализацию — datetime нужно конвертировать в ISO строку"
**Результат:** Работает. Тесты проходят.

### Итого

- Количество промптов: 2
- Что пришлось исправлять вручную: ничего
- Время: ~10 мин
